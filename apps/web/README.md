# Web Frontend Template

Template inicial de frontend para o SaaS com visual de dashboard, layout responsivo e componentes base.

## Run
```bash
npm install
npm run dev
```

## Estrutura
- `app/layout.tsx`: layout global
- `app/page.tsx`: tela inicial
- `components/dashboard-template.tsx`: template base
- `app/globals.css`: design tokens e estilos
